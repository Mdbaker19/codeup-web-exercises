Line breaks and horizontal rules

  • Block level, void elements
  • Both create a line break in the documents
    1. < hr > does so with a horizontal line
      • The < hr > element is most often displayed as a horizontal rule
        that is used to separate content (or define a change) in an HTML page.
    2. < br > does so with a line break
      • The < br > element has a single, well-defined purpose — to create a line break in a block
        of text. As such, it has no dimensions or visual output of its own, and there
        is very little you can do to style it.
